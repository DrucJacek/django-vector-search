# Mini Vector Search

## About The Project
This repository contains a lightweight Python script for semantic text search using **Sentence Transformers**. 

Instead of relying on traditional, exact-keyword matching, this script converts text into vector embeddings. This allows the search to understand the actual meaning and context of a user's query, providing much more accurate and relevant results.

Currently, this is a standalone utility script. In the future, this logic will be integrated into a **Django / Django REST Framework (DRF)** project to power the search functionality of a blog platform.
